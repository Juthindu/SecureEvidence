    Welcome to **SecureEvidence**, a state-of-the-art web application designed to manage and secure evidence for criminal cases using blockchain technology and the InterPlanetary File System (IPFS). Our platform ensures that evidence remains tamper-proof, immutable, and accessible only to authorized users. By leveraging these advanced technologies, SecureEvidence provides a transparent and decentralized system that guarantees the integrity and traceability of evidence. The application supports various user roles, including Super Admin, Level One (Admin), and Level Two, each with specific permissions to facilitate secure and efficient management. Whether you're adding new evidence, managing cases, or updating user roles, SecureEvidence ensures that all operations are conducted with the highest level of security and accuracy.
